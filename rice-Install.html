RICEDROID INSTALL

1)  Follow all the preinstall steps from here.
2)  Install OrangeFox using the guide here, or TWRP (3.7.0_12-2 and later versions) using the guide here.
3)  Download the riceDroid build for your device A52 & A72. Gapps are included.
4)  Flash the riceDroid zip, either from sdcard or usb storage, or use adb sideload method.
5)  Format data in OrangeFox/TWRP, otherwise you may end up in a bootloop if using encryption or coming from stock ROM.

7)  Reboot.
