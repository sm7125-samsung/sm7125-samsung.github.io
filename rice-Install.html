RICEDROID INSTALL

  1. Follow all the preinstall steps from here.
  2. Install OrangeFox using the guide here, or TWRP using the guide here.
  3. Download the riceDroid build for your device from here. Google Apps are included.
  4. Copy the riceDroid zip to your phone's SD card or a USB Storage, or use adb sideload method.
  5. Flash the riceDroid zip using the "Install" button menu.
  6. After that go to "Wipe", "Format data" and confirm with "yes", otherwise you may end up in a bootloop if using encryption or coming from stock ROM.
  7. Again go into the "Wipe" menu, select "Advanced Wipe" and tick metadata and cache partitions.
  8. If you did everything correctly then your phone will reboot into riceDroid.
