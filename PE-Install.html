PIXEL EXPERIENCE INSTALL

  • PE Recovery
  • TWRP/OFox
  1. Follow all the preinstall steps from here.
  2. Download the PixelExperience Recovery (A52) (A72). Simply download the latest recovery file.
  3. Rename the downloaded image to “recovery.img”, open a Command Prompt in the folder where the file was downloaded, and TAR the file by running tar -cvf recovery.tar recovery.img.
  4. Power off the device, and boot it into download mode:With the device powered off, hold Volume Up + Volume Down and connect USB cable to PC and device.
  5. Now, click the button that the onscreen instructions correlate to “Continue”.
  6. Open Odin and select the recovery.tar from step 2 in the AP slot. Click start.
  7. Download the PixelExperience installation package (A52) (A72) that you would like to install.
  8. If you are not in recovery, reboot into recovery: With the device powered off, hold Volume Up + Power while the device is connected to a PC via USB cable.
  9. Now tap Factory Reset, then Format data / factory reset and continue with the formatting process. This will remove encryption and delete all files stored in the internal storage, as well as format your cache partition. Return to the main menu.
  10. Sideload the PixelExperience .zip package:
  11. On the device, select “Apply Update”, then “Apply from ADB” to begin sideload.
  12. On the host machine, sideload the package using: adb sideload filename.zip
  13. Once you have installed everything successfully, click the back arrow in the top left of the screen, then “Reboot system now”.
  1. Follow all the preinstall steps from here.
  2. Install OrangeFox using the guide here, or TWRP (3.7.0_12-2 and later versions) using the guide here.
  3. Download the Pixel Experience build for your device (A52) (A72). Gapps are included.
  4. Flash the Pixel Experience zip, either from sdcard or usb storage, or use adb sideload method.
  5. Format data in OrangeFox/TWRP, otherwise you may end up in a bootloop if using encryption or coming from stock ROM.
  6. Wipe metadata and cache partitions. If metadata partition isnt available in wipe menu you are on ​VERY outdated TWRP/OFox.
  7. Reboot.
