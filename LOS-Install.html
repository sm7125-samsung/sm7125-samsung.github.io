LINEAGE OS INSTALL

  • LOS Recovery
  • TWRP/Ofox
  1. Follow all the preinstall steps from here.
  2. Download the LineageOS zip, recovery.img and vbmeta.img from here: A52 | A72. Simply download the latest files.
  3. If wanted, download Google apps too
  4. Open a Command Prompt in the folder where the files were downloaded, and TAR each .img file by running
    tar -cvf recovery.tar recovery.img and tar -cvf vbmeta.tar vbmeta.img. Alternatively, you can use 7-Zip by doing Right Click on each .img file, then select "Add to Archive" in the 7-Zip menu and select "Tar" as compression method.
  5. Power off the device and connect it to a PC.
  6. Now hold Volume Up + Volume Down + Power buttons until you enter Download Mode.
  7. Now, confirm by pressing Volume Up once.
  8. Open Odin and make sure your phone is detected (it says "Added!" in the Message window).
  9. Put the recovery .tar file you created into the AP tab.
  10. Put the vbmeta .tar file you created into the USERDATA tab.
  11. Once the phone reboots after pressing "Start", hold Volume Up + Power buttons until you reach LineageOS recovery.
  12. Click "Start", the phone will automatically reboot. Now hold Volume Up + Power buttons for at least 10 seconds until you boot into LineageOS recovery.
  13. If you properly followed all the steps you should now see the LineageOS recovery menu. If not, repeat all steps.
  14. Now click Factory Reset and confirm the process.
  15. Now select "Apply Update" and then "Apply from ADB" on the phone to sideload the LineageOS .zip package. Alternatively, copy the LineageOS zip to your phone's SD card and select "Choose from SD card".
  16. On your Computer, sideload the LineageOS package using adb sideload lineageOS.zip
  17. Optionally, flash Gapps right after: adb sideload MindTheGapps.zip.
  18. Once you have installed everything successfully, click the back arrow in the top left corner of the screen, then “Reboot system now”.
  19. If you did everything correctly then your phone will reboot into LineageOS.
  1. Follow all the preinstall steps from here.
  2. Install OrangeFox using the guide here, or TWRP using the guide here.
  3. Download the LineageOS build for your device A52 | A72.
  4. If wanted, download Google apps too
  5. Copy the LineageOS zip to your phone's SD card or a USB Storage, or use adb sideload method.
  6. Flash the LineageOS zip using the "Install" button menu.
  7. After that go to "Wipe", "Format data" and confirm with "yes", otherwise you may end up in a bootloop if using encryption or coming from stock ROM.
  8. Again go into the "Wipe" menu, select "Advanced Wipe" and tick metadata and cache partitions.
  9. If you did everything correctly then your phone will reboot into LineageOS.
