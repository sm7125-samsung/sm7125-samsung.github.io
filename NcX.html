NcX

NcX is a Stock OneUI 5 based mod that bings in a lot more features to the stock, by porting them from S21.
INSTALL GUIDE
Supported Devices
  • SM-A525F
  • SM-A525F/DS
  • SM-A525M
  • SM-A525M/DS
Extra Features
  • Based on S21 FE May firmware (G990BXXS4EWE3)
  • Bootanimation zip support instead of Samsung's QMG implementation
  • Samsung DeX support
  • Advanced Reboot options (Reboot to recovery, to download, ...)
  • Native AppLock
  • Scan text via Camera
  • OutDoor mode
  • Fixed Samsung Health
  • Fixed Private Share and Secure Wi-Fi
  • Live Blur in Quick Settings
  • FM Radio
  • Remaster Picture in Gallery
  • Allow screenshots on secure apps
  • Secure Folder fixed
  • Object eraser
  • Shadow and Reflection eraser
  • Debloated
  • Edge Lighting fully enabled
  • High-End Animations
  • Enhanced pocessing
  • CSC Features like - Stock call recording, Speed meter on statusbar, Spam call blocker, etc.
  • Added camera privacy toggle to remove camera access across apps
  • Added HDR10+ support
  • Added optional AOD to clock animation. Avoid using it with FOD. You can disable it in the installer.
Known Bugs

No known issues!
