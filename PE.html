PixelExperience

PixelExperience is an AOSP based ROM, with Google apps and the most Pixel features included, for example Pixel Launcher, Wallpapers, Icons, Fonts, Bootanimation and many more.
INSTALL GUIDE
Supported Devices
  • SM-A525F
  • SM-A525F/DS
  • SM-A525M
  • SM-A525M/DS
  • SM-A725F
  • SM-A725F/DS
  • SM-A725M
  • SM-A725M/DS
Known Bugs
  • No VoLTE/VoWIFI support (Samsung's implementation doesn't work on AOSP)
  • Various camera limitations (no 1080p@60FPS, 4k recording and 64MP support)
  • Camera apps that use the CameraX framework produce ~70mb JPEGs
